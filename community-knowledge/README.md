# Community Knowledge

This directory contains publishable examples for hosted Power profiles.

Current pieces:

- `registry/profile-registry.v1.json`
  A static registry index for online profile discovery.
- `profiles/app-horse/app-horse.power-profile.v1.json`
  A hosted sample profile package for `App Horse`.

These files are designed to be:

- readable by humans
- easy to publish on GitHub Pages or raw GitHub
- compatible with the current portable profile shape
- future-friendly for URL-based profile import
