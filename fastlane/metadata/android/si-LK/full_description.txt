ආරක්ෂිත, විනිවිද බවක් ඇති සහ සුරක්ෂිත Android පද්ධතියකට ආයුබෝවන් කියමූ.

VPN: ඔබ කැමති බහු (multiple) WireGuard VPNs වෙත සම්බන්ධ විය හැක.

ආරක්ෂිතයි: ස්ථාපිත යෙදුම්වල network ක්‍රියාකාරකම් හෙළි කරන අතර ඒවා අක්‍රිය කිරීමට, ඉවත් කිරීමට හෝ firewall කිරීමට ඔබට ඉඩ සලසයි.

සුරක්ෂිතයි: පුද්ගලික තොරතුරු සොරකම් කරන, ගිණුම් අත්පත් කර ගන්නා spyware, ransomware සහ malware වලට එරෙහිව පළමු ආරක්ෂක පෙළක් සේ ක්‍රියාකරයි. වෙබ් අඩවි, සමාජ මාධ්‍ය සහ messaging apps වලට භාවිතා කරන DNS manipulations වලින් ආරක්ෂා කරයි.

වේගවත්: ඔබේ අන්තර්ජාලය හැකි තරම් වේගවත් සහ ඉහළ ස්ථාවරයක පවතින බව සහතික කිරීම සඳහා ලොව පුරා ස්ථාන 300+ කට යොදවා ඇති DNS Servers.

විවෘතයි: විවෘත මූලාශ්‍ර (Open-source) සහ trackers වලින් තොරයි.

***විශේෂාංග***

Firewall: Apps අන්තර්ජාලයට සම්බන්ධ වීම වැලැක්වීමට කටයුතු කරයි. Spyware , malware හා ransomware ඇතුළුව තවත් දේ අවහිර කරයි.

WiFi හෝ mobile data හරහා අන්තර්ජාලයට සම්බන්ධ වන ඕනෑම යෙදුමක් firewall එකකින් අවහිර කරයි. බොහෝ ආකාරයේ අනිසි නිරීක්ෂණ (surveillance) සඳහා දත්ත, network එකකට යැවීම අවශ්‍ය වන බැවින්, ඒවා firewall කිරීම මගින් තර්ජනය ඵලදායී ලෙස අවම කරයි (File Managers, Alarm Clock හා Calculator එසේ ක්‍රියාත්මක කිරීමට කිසිදු අන්තර්ජාල ප්‍රවේශයක් අවශ්‍ය නොවන යෙදුම් සඳහා උදාහරණ කිහිපයකි). Firewall විශේෂාංගය පසුබිම් යෙදුම් හඳුනා ගැනීමට සහ firewall කිරීමට ප්‍රවේශ්‍යතා සේවාව (Accessibility Service) භාවිතා කරයි.

අධීක්ෂණය: පැමිණෙන සහ පිටතට යන අන්තර්ජාල ගමනාගමනය පිළිබඳ තොරතුරු තබාගනී.

සක්‍රීය කර ඇත්නම්, සම්බන්ධතා ලොග් (connectivity logs) එකතු කර විශ්ලේෂණය කරනු ලැබේ. ස්වයංක්‍රීය වාර්තා කලින් නොදන්නා හෝ සැක සහිත සම්බන්ධතා flag කරන අතර spyware ජාල මගින් දත්ත සොරකම් කිරීමට දරන උත්සාහයන්ගේ ප්‍රමාණ හෙළි කරයි. අපගේ සමීක්ෂණ වලදී, ගමනාගමනයෙන් 60% ක් පමණ සලකුණු කර දන්නා spyware මගින් ආරම්භ කරන ලද බවට වාර්තා කර ඇත. ජනනය කරන ලද ලොග් පරිශීලකයාට ඔවුන්ගේම විශ්ලේෂණය සඳහා real-time තබාගත හැක.

WireGuard: ආරක්ෂිත පිටතට යන අන්තර්ජාල ගමනාගමන.

මෙම යෙදුම ඔබ කැමති ඕනෑම අනුකූල VPN සැපයුම්කරුවෙකුට සම්බන්ධතා encrypt කිරීමට වේගවත්, නවීන, සැහැල්ලු සහ ආරක්ෂිත WireGuard ප්‍රොටෝකෝලය භාවිතා කරයි.

සමාජ-වාරණයට එරෙහි වීම:

මෙම යෙදුම ඔබගේ අන්තර්ජාල සම්බන්ධතාවය ලොව පුරා අපගේ ස්ථාන 300+ ක පැවැත්මක් සහිතව Rethink විසින් පවත්වාගෙන යනු ලබන DNS server එකකට encrypt කරයි. DNS server එකක් යනු අන්තර්ජාලයේ ලිපින පොතයි: DNS server එකක්, ඔබට වෙබ් අඩවියකට පිවිසීමට හෝ යෙදුමක් විවෘත කිරීමට අවශ්‍ය නිශ්චිත ලිපින සපයයි.

දත්ත නිරීක්ෂණය (Data Monitoring): යෙදුමකට සහ සම්බන්ධතාවයකට Mobile data හෝ WiFi දත්ත භාවිතය නිරීක්ෂණය කල හැක.
